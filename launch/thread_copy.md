Anthropic found a workspace inside Claude.

OpenLine governs what gets to leave it.

That is the bridge.

J-space gives a way to inspect what a model may be silently considering.

But inspection is not governance.

The hard question is what happens when an internal thought becomes a tool call, memory write, file edit, handoff, refusal, or external action.

So I built a small prototype:

OpenLine J-space Commit Bridge.

Workspace readout enters a wide buffer.

Candidate actions pass through a strict witness quorum.

Every commit or quarantine leaves a receipt.

A workspace helps the model think.

A receipt helps the world hold the crossing accountable.
